# diy-sip-and-puff-system
Open-Source Sip &amp; Puff Maussteuerung für Menschen mit eingeschränkter Fingermotorik. Arduino-basiertes System mit Drucksensor (Saugen/Blasen für Klicks) und Joystick. Moderne Python-GUI zur Echtzeit-Konfiguration. Entwickelt im Rahmen eines Human Factors Projekts an der Hochschule Furtwangen.
